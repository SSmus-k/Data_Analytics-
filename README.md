This project is a desktop-based healthcare analytics dashboard designed to help doctors, nurses, and hospital administrators visualize patient ata and identify important clinical and operational trends.

## Technologies Used
- Python
- Pandas
- NumPy
- Matplotlib
- Tkinter

## Installation Steps

1. Clone the repository or download the ZIP file

git clone <https://github.com/SSmus-k/Data_Analytics->
cd Data_Analytics

2. Install dependencies

pip install -r requirements.txt

If multiple Python versions exist:

python -m pip install -r requirements.txt

3. Run the application

python app.py

## Features
- Cleans noisy patient data
- Displays corrected data in table form
- Visualizes patient status distribution
- Visualizes department load
- Computes average hospital stay
- Runs fully offline (no browser)

## Intended Users
- Doctors
- Nurses
- Hospital administrators
- Healthcare analysts
